## 初期費用：だいたい5〜15万円

猫を迎えるときに最初にかかるお金です。迎え方・猫のサイズ・選ぶグッズによって幅があります。

### 猫の入手費用

| 迎え方 | 費用の目安 |
|--------|------------|
| 保護猫・里親譲渡 | 0〜3万円（ワクチン・去勢込みの場合も） |
| 保護団体からの引き取り | 1〜5万円 |
| ペットショップ | 5〜30万円（品種による） |

**ズボラ的には保護猫が最強です。** 初期費用が安いだけじゃなく、ある程度育っていてトイレを覚えていることも多い。子猫ほど手がかからないのも一人暮らしには助かります。

### 必要なグッズ（初期購入）

| グッズ | 目安金額 |
|--------|----------|
| キャリーバッグ | 3,000〜1万円 |
| トイレ本体 | 1,000〜5万円（自動か手動かで大きく違う） |
| ご飯皿・水飲み | 500〜5,000円 |
| 爪とぎ | 500〜2,000円 |
| おもちゃ | 1,000〜3,000円 |
| ケージ（任意） | 5,000〜3万円 |

**初期グッズの合計目安：2〜10万円**

トイレを自動にするかどうかで金額が大きく変わります。最初から自動トイレを選ぶと初期費用は上がりますが、毎日のお世話コストと精神的なラクさで十分回収できます。

### 初回の医療費

| 項目 | 目安金額 |
|------|----------|
| 初回ワクチン（2〜3回） | 6,000〜1.5万円 |
| 去勢手術（オス） | 1.5〜3万円 |
| 避妊手術（メス） | 2〜5万円 |
| 健康診断 | 3,000〜8,000円 |

去勢・避妊手術は必須ではありませんが、**健康面・行動面・においの面**でほぼ全飼い主が実施しています。一人暮らしでは特に、発情期の鳴き声問題が大きいので早めにやっておいて損なし。

## 毎月かかる費用：月8,000〜2万円

日常的にかかる費用です。猫の頭数・フードのグレード・医療費の有無で変わります。

### フード代

| フードの種類 | 月あたりの目安 |
|--------------|----------------|
| ドライフード（市販品） | 2,000〜4,000円 |
| ドライフード（プレミアム） | 4,000〜8,000円 |
| ウェットフード追加 | ＋1,000〜3,000円 |

一人暮らしの猫1匹なら、月**3,000〜6,000円**が現実的なラインです。

フード選びは「とにかく安く」より「品質と値段のバランス」が長期的には安上がり。粗悪なフードを長年食べると病気になりやすく、医療費がかさむケースもあります。

### 猫砂代

| 猫砂の種類 | 月あたりの目安 |
|------------|----------------|
| 紙製・木製 | 1,000〜2,000円 |
| 鉱物系（ベントナイト） | 1,500〜3,000円 |
| シリカゲル | 1,000〜2,500円 |

自動トイレを使っている場合は専用シートの費用になります（月500〜2,000円程度）。

👉 [全自動トイレに使える猫砂おすすめ3選【ほこり・においを正直に比較】](/posts/cat-litter-auto-toilet/)

### 医療費の積み立て

健康な猫でも、年1回のワクチンとノミ・マダニの予防薬は必要です。

| 項目 | 年間目安 | 月割 |
|------|----------|------|
| ワクチン（年1回） | 5,000〜8,000円 | 約400〜700円 |
| ノミ・ダニ予防 | 5,000〜1.5万円 | 約400〜1,200円 |
| 突発的な医療費の積み立て | — | 3,000〜5,000円推奨 |

猫は**急に体調を崩すことがある**ので、医療費の積み立てはズボラでも絶対やっておいた方がいいです。うちも1匹が腎臓の数値が悪くなって月1回通院になったとき、積み立てていて助かりました。

## 年間費用のざっくり計算

| 費用項目 | 年間目安 |
|----------|----------|
| フード | 36,000〜96,000円 |
| 猫砂 | 12,000〜36,000円 |
| ワクチン・予防薬 | 10,000〜23,000円 |
| 医療費（突発含む） | 0〜10万円 |
| おやつ・グッズ | 12,000〜36,000円 |
| **合計（目安）** | **7〜26万円** |

「幅が広すぎ！」って思いますよね。でもこれが現実で、**医療費次第で大きく変わる**のが猫を飼う費用の正直なところです。

健康な猫が続けば年7〜10万円で収まることも十分ありますが、病気になると一気に跳ね上がります。

## ズボラ的「節約できるポイント」3つ

### ① お世話を自動化してサボりつつ猫を守る

「ズボラ＝お世話が雑」ではありません。**自動化することで、逆に猫の健康管理が上がることが多い。**

たとえば：
- **自動給餌器**→ 定時定量で太りすぎ・空腹を防ぐ
- **自動給水器**→ 水を飲む量が増えて腎臓・泌尿器を守る
- **全自動トイレ**→ 排泄の変化にすぐ気づける
- **ロボット掃除機**→ 猫の毛・猫砂を毎日自動で除去

## 一人暮らしで猫を飼うときの「現実的な確認リスト」

費用以外にも、一人暮らしで猫を飼う前に確認しておきたいことをまとめます。

- [ ] 賃貸の場合、ペット可物件か確認した
- [ ] 長期不在時の預け先（ペットホテル・知人）が確保できる
- [ ] 月1〜2万円の出費を継続できる見通しがある
- [ ] 緊急時の医療費（最低5万円）を用意できる
- [ ] 猫の寿命（15〜20年）まで責任を持てる

**特に「長期不在時の対応」は重要です。** 仕事の出張・旅行など数日以上家を空ける場合どうするか、飼い始める前に決めておくと安心です。

## まとめ：一人暮らしの猫の費用、月1万円あれば何とかなる

| | 最低ライン | 余裕を持つなら |
|---|------------|----------------|
| 初期費用 | 5〜7万円 | 10〜15万円 |
| 月々の費用 | 8,000〜1.2万円 | 1.5〜2万円 |
| 緊急医療費の備え | 5万円 | 10〜20万円 |

正直に言うと、「とにかく安く飼えるか？」という問いには「ある程度かかります」が答えです。

でも、**月1〜1.5万円を続けられる見通しがあれば十分飼えます。** 保護猫を迎えて、フードは品質重視で選んで、お世話は自動化でラクをすれば、費用も体力も無理なく続けられます。

猫がいる暮らしは、それだけの価値が絶対あります。
